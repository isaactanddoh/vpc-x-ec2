### Project Title
VPC and EC2 Terraform Project

### Project Description
This project involves creating a Virtual Private Cloud (VPC) and Elastic Compute Cloud (EC2) instances on Amazon Web Services (AWS) using Terraform as Infrastructure as Code (IaC). The goal is to automate the setup of AWS infrastructure, ensuring consistent and repeatable deployments.

### Prerequisites
- AWS account
- Terraform installed
- AWS CLI configured
- Basic knowledge of AWS and Terraform

### Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/isaactanddoh/vpc-x-ec2.git
   cd vpc-x-ec2
   ```
2. **Initialize Terraform:**
   ```bash
   terraform init
   ```

### Usage
1. **Configure AWS credentials:**
   Ensure your AWS CLI is configured with the appropriate credentials.
2. **Create a Terraform plan:**
   ```bash
   terraform plan
   ```
3. **Apply the Terraform plan:**
   ```bash
   terraform apply
   ```
4. **Destroy the infrastructure:**
   ```bash
   terraform destroy
   ```
